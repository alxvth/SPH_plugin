# SPH Plugin
[ManiVault](https://github.com/ManiVaultStudio) plugin for [Superpixel Hierarchies and Embeddings for High-Dimensional Images](https://github.com/alxvth/SPH).

```
git clone --recurse-submodules git@github.com:alxvth/SPH_plugin.git
```

## Building
ManiVault related: define `ManiVault_DIR` pointing to the ManiVault CMake files (in `cmake/mv` of your ManiVault install path).

See [SPH](https://github.com/alxvth/SPH) for further build instructions of the underlying library.
